项目描述：利用遗传算法实现了对TSP问题的求解，本例中所求解的问题是gr48问题，用时12秒左右，可求得最优解。
文件描述：
	GA.cpp：实现遗传算法，具体实现种群初始化、求解适应度、交叉、变异、轮盘赌选择等算法。
	GA.h：遗传算法的参数设置，包括基因所占位数、父代种群大小、子代种群大小、交叉和编译因子、迭代次数等参数设置。
	main.cpp：主程序框架。
	tools.h：主要实现一些工具函数，例如两个整数交换顺序、排序等函数。
	gr48.txt：数据集。分别代表两个城市之间的距离，例如第一个数为城市1->1距离，第二个数字2->1距离，第三个数字2->2距离，第四数字3->1距离，如此类推，一直到最后一个数字表示城市48->48之间的距离（为0）。
	README：代码说明文件。
